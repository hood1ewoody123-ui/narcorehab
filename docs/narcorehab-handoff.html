01 · Введение

Для команды разработки

Документ собран из рабочего прототипа. Импорт в Figma — через HTML-плагин; слои размечены секциями. Исходный код — ориентир для точных значений.

Ограничения продукта (обязательны): без красного цвета · без восклицательных знаков в копирайте · без агрессивных CTA · тон деликатный, снимающий страх.
